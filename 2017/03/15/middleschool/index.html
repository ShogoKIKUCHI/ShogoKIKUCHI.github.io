March 15, 2017

中学生ピロリ菌検査実施マニュアル案

本マニュアル（案）は、自治体が地元の医療機関と協力して、中学生のピロリ菌検査を実施する場合の手順や留意点を示したものである。これから、同事業を実施する場合に、参考にしていただければ幸いである。実際に使用された文書を付してあるので、こちらも参考にしていただければ幸いである。特に重要なのは、1次検査で陽性とされた生徒に、どのような形で2次検査や治療を行うかという点と、除菌に伴って副反応（副作用）が起きた場合の医療費を含めた対処である。 未だ不備な点もあると思われるので、ご指摘いただければ幸いである。

１． 担当部署の決定 関連部署 自治体内部 ・学校健診担当部門（教育委員会内の部署など） ・健康対策部門（健康課、保健センター、衛生部など） 学校 ・中学校長会 ・養護教諭の連絡会議 ・学校医 医師会 ・公衆衛生担当理事など

解説：自治体で、どの部署が担当するかを決める必要がある。学校を通じて実施する場合は、学校健診を担当する部署が、各学校との連絡調整にあたる必要がある。保護者からの問い合わせや説明については、保健センターなど健康対策を行っていて保健師や医師などの専門職がいる部署でないと実際困難である。保健センターと医師会（担当理事など）で保護者へ説明や対応について協力関係を予め作っておくことが望ましい。理想的には、まず中学（相当の教育機関）の長、各校の養護教諭などへの説明の場を設けるべきである。保護者への説明については、説明会を行うことが望ましい。多数の出席は望めない場合は、わかりやすい文書の配布で替えてもよい。各学校の校医への周知も不可欠である。

２． 計画の策定と実施 ・保護者への通知方法、説明内容を具体的に決める。 　法定健診ではない段階での実施では検査することについて拒否の機会を設けるか同意を得て実施することが望ましい。（方法として、ピロリ検査だけに検体を収集する場合には検体の不提出で可能であるが、一般尿検査の検体を用いる場合には、拒否のないもしくは同意のある検体のみの検査を行う必要がある。）

・通知方法（郵送／学校配布） 　中学生が対象の場合、学校配布だと保護者に届かないことがあるので、多少経費はかかるが、郵送が望ましい。

・検体の収集方法 　医療機関などでの収集では高い検査参加率は得られない。 学校での収集では高い検査受診率が得られるため、学校を通じての収集が望ましい。

・1次検査の方法 　侵襲のない（痛みや危険を伴わない）方法には、尿中抗体検査と便中抗原検査がある。学校健診で検尿が行われていることや、尿の方が扱いやすく提出率が高いことから尿中抗体検査が採用されることが多い。尿検査では、目視で判定するイムノクロマト法と検査機関で検査するELISA法があるが、精度管理の面からはELISAの方が望ましい。

・1次検査の委託先 　収集を学校で行う場合は、学校健診の依頼先と同様の形で委託することになる。 　医療機関で収集を行う場合も、精度管理の面から同一の検査機関に委託する形をとることが望ましい。

・結果の判定方法と通知の方法 　判定の基準値は、偽陰性（見逃し）を減らすために、能書の基準値よりやや低めにとることが望ましく、実際には2次検査の費用などを勘案して決める必要がある。 　結果の通知は、生徒同士が結果を見せ合うことを避けるため、保護者宛に郵送で実施することが望ましい。この場合の通知の内容も、陽性、陰性それぞれについて、保護者への説明文書の内容を予め決めておく必要がある。 　 ・2次検査の方法と実施方法、費用負担 　わが国の中学生の尿検査陽性率は5%以下という報告が多く、5%を大きく越えることはない。尿が１次検査である場合、尿素呼気試験を、事情によって便中抗原検査を2次検査として実施すべきである。2次検査をどこで行うか、また１件あたりの費用はいくらにするかを決める必要がある。医師会加入の医療機関の中から依頼先を決める場合と、専門の医療機関に限定して委託する場合がある。自治体の費用で実施するのか、個人負担かなど費用負担も予め決める必要がある。

・2次検査で陽性とされた生徒の扱い ピロリ菌治療を手がける専門の医療機関での診療が望ましい。

[除菌の必要性] 　腹痛などの腹部症状がある場合は早めに除菌が必要であり、状況によって内視鏡検査が必要である。ピロリ菌が鉄欠乏性貧血の原因になることが知られているので、鉄欠乏性貧血がある場合も早めの除菌が必要である。中学生の鉄欠乏性貧血はピロリ菌が関係していることが多い。 無症状の場合でも、胃がんの予防を図るために除菌する必要がある。直接的な根拠となる研究成果はないが、これまでのいくつかの研究から研究胃がん予防の面からは除菌は早い方がよいとされている。将来の児への家族内感染を防ぐためには、遅くとも児ができる前（実際には女性では妊娠前、男性では児の出生前）に除菌が完了していることが望ましい。

　 [除菌治療] 　　　除菌は小児でも成人でも、3剤すなわちプロトン・ポンプ阻害剤（以下PPI）、アモキシシリン（ペニシリンの一種）に加え、クラリスロマイシンもしくはメトロニダゾールを1週間服用する。クラリスロマイシンは耐性菌が多いため70%前後の成功率であるが、メトロニダゾールでは95%前後の成功率である（健康保険＝成人ではクラリスロマイシンを含む3剤でまず行い、不成功の場合にメトロニダゾールを含む3剤で治療することになっている）。培養用の胃液を採取することで薬剤耐性（特にクラリスロマイシン耐性）は検査可能であるが、検体採取に苦痛が伴うことや培養のコストがかかる。 除菌が成功したかどうかは、3剤1週間の服薬終了後4週以降に尿素呼気試験か便中抗原検査で確認する。血清や尿中の抗体検査は除菌成功後も長期間高値が続くので除菌判定には用いない。

　　[除菌の安全性] 中学生は小児とされるので、現時点でのPPIの使用は厳密には健康（医療）保険で認められていない。海外では、安全性が確認されているので、わが国でも、保険適用を関係学会が働きかけているところである。 これまで最も重篤な除菌の副作用はペニシリン・アレルギーによるとされている。除菌治療にあたっては、ペニシリン・アレルギーについてよく聞いた上で薬剤を選ぶ必要がある。わが国の小児消化器を治療している専門医などに対して行ったアンケート調査では、これまで約600例の除菌で大きな副作用の事例はない。

盛夏の候、保護者の皆様におかれましてはますますご健勝のことと存じます。 平素は、本市行政及び教育にご理解、ご協力を賜っておりますこと、厚くお礼申し上げます。 　本市は胃がんの死亡数も県平均に比べて多く、より早い段階での予防を重要な対策として取り組むことが必要となってきております。 そこで、市独自の胃がん予防対策として、本年度より、市内の中学〇年生全員を対象に、ピロリ菌の感染の有無を調べる検診（尿検査）を実施いたします。 実施の詳細については別紙（黄色の用紙）に詳しく掲載しておりますので、必ず熟読くださいますようお願い申し上げます。 なお、検査を希望しない場合は、期日までに健康課までご連絡ください。検査対象から除外させていただきます。 本検診内容やピロリ菌等についてのご質問やご相談も、健康課で承っておりますので、不明の点はお手数ですが、電話等にてお問い合わせください。
